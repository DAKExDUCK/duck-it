# DUCK-IT

## Setup

### Linux

Clone the repository:

```sh
$ git clone https://github.com/DAKExDUCK/duck-it.git
$ cd duck-it
```

Create venv and activate it:

```sh
$ python3 -m venv venv
$ source venv/bin/activate
```

Then install the dependencies:

```sh
(venv) $ pip install -r requirements.txt
```
